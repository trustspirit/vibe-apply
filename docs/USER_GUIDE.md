## 지원서 제출 시나리오

지원자가 지원서를 제출하면 다음과 같은 전체 프로세스가 진행됩니다.

### 1단계: 지원서 작성 및 제출

**지원자 (Applicant)**
- 지원서 작성 페이지에서 개인 정보 입력
  - 이름, 이메일, 전화번호, 성별, 나이
  - 스테이크/지방부 및 와드/지부 선택
  - 선교 경험 여부
  - 추가 정보 (선택사항)
- 초안 저장 (DRAFT 상태)
  - 언제든지 수정 가능
  - 제출 전까지 저장 가능
- 지원서 제출 (AWAITING 상태)
  - 제출 후에는 관리자 검토 전까지 수정 가능
  - 이미 추천서가 있는 경우 제출 불가 (시스템에서 자동 체크)

**시스템 동작**
- 이메일, 스테이크, 와드로 기존 추천서 확인
- 이미 추천서가 있으면 제출 차단 및 오류 메시지 표시
- 제출 성공 시 상태가 `AWAITING`으로 변경

### 2단계: 리더의 지원서 확인 및 추천서 작성

**감독 (Bishop) - 승인된 리더**
- 리더 대시보드에서 본인 와드의 지원서 확인
- 추천서 관리 페이지에서 지원서 목록 확인
  - 지원서는 "지원됨" 태그와 함께 표시
  - 지원서 상세 정보 확인 가능
- 지원서에 대한 추천서 작성
  - 지원서 상세 화면에서 "추천하기" 버튼 클릭
  - 지원서 정보가 자동으로 추천서 폼에 채워짐
  - 추가 정보 입력 후 추천서 제출
  - 추천서는 자동으로 해당 지원서와 연결됨 (`linkedApplicationId`)
  - 추천서 상태는 `SUBMITTED`로 설정됨
- 지도자 코멘트 작성
  - 지원서 또는 추천서에 내부 코멘트 추가
  - 다른 지도자와 정보 공유 (지원자는 볼 수 없음)

**스테이크 회장 (Stake President) - 승인된 리더**
- 리더 대시보드에서 본인 스테이크 전체의 지원서 확인
- 추천서 관리 페이지에서 지원서 목록 확인
- 지원서에 대한 추천서 작성 (감독과 동일한 프로세스)
- 지도자 코멘트 작성

**시스템 동작**
- 지원서 제출 시 해당 스테이크/와드의 리더에게 표시
- 추천서 작성 시 이메일 매칭으로 자동 연결
- 추천서와 지원서가 연결되면 UI에서 하나의 항목으로 통합 표시
  - 지원서 항목에 "추천됨" 태그 표시
  - 연결된 추천서는 별도 항목으로 표시되지 않음

### 3단계: 관리자 검토

**관리자 (Admin) / 대회 지도자 (Session Leader) - 승인된 리더**
- 관리자 대시보드에서 전체 통계 확인
  - 전체 제출 수, 검토 대기 중, 승인된 항목 수
  - 오늘 제출된 항목 수
- 신청서 검토 페이지에서 검토
  - 전체, 대기 중, 승인, 거부 탭으로 필터링
  - 지원서와 추천서를 함께 확인
  - 각 항목의 상세 정보 확인
    - 지원서 정보
    - 연결된 추천서 정보 (있는 경우)
    - 지도자 코멘트 (읽기 전용)
- 승인/거부 결정
  - 지원서 상태 변경: `AWAITING` → `APPROVED` 또는 `REJECTED`
  - 추천서 상태 변경: `SUBMITTED` → `APPROVED` 또는 `REJECTED`
  - 승인된 지원서는 CSV로 내보내기 가능

**시스템 동작**
- 상태 변경 시 즉시 반영
- 승인/거부 후에는 해당 항목 수정 불가
- 승인된 지원서는 통계에 반영

### 4단계: 결과 확인

**지원자 (Applicant)**
- 지원서 페이지에서 최종 상태 확인
  - `APPROVED`: 승인됨
  - `REJECTED`: 거부됨
- 승인/거부 후에는 지원서 수정 불가
- 지도자 코멘트는 볼 수 없음

**감독 (Bishop) / 스테이크 회장 (Stake President)**
- 추천서 관리 페이지에서 추천서 상태 확인
  - `APPROVED`: 승인됨
  - `REJECTED`: 거부됨
- 승인/거부 후에는 추천서 수정 불가
- 지도자 코멘트는 계속 작성 및 확인 가능

### 시나리오별 워크플로우

#### 시나리오 A: 지원서만 제출된 경우
1. 지원자 → 지원서 제출 (`AWAITING`)
2. 감독/스테이크 회장 → 지원서 확인
3. (선택) 감독/스테이크 회장 → 추천서 작성 및 제출 (`SUBMITTED`)
4. 관리자/대회 지도자 → 지원서 및 추천서(있는 경우) 검토
5. 관리자/대회 지도자 → 승인/거부 결정

#### 시나리오 B: 추천서가 먼저 작성된 경우
1. 감독/스테이크 회장 → 추천서 작성 및 제출 (`SUBMITTED`)
2. 지원자 → 지원서 제출 시도
3. 시스템 → 이미 추천서가 있음을 감지하고 제출 차단
4. 지원자 → 감독 또는 스테이크 회장에게 문의 필요

#### 시나리오 C: 지원서와 추천서가 모두 있는 경우
1. 지원자 → 지원서 제출 (`AWAITING`)
2. 감독/스테이크 회장 → 지원서 확인 및 추천서 작성 (`SUBMITTED`)
3. 시스템 → 추천서와 지원서 자동 연결
4. UI → 하나의 항목으로 통합 표시 ("지원됨" + "추천됨" 태그)
5. 관리자/대회 지도자 → 통합된 정보로 검토
6. 관리자/대회 지도자 → 승인/거부 결정

## 역할별 기능 상세

## 지원자 (Applicant)

### 접근 가능한 기능

#### 지원서 작성 및 제출
- **개인 정보 입력**
  - 이름 (필수)
  - 이메일 (필수)
  - 전화번호 (필수)
  - 성별 (필수): 남성, 여성
  - 나이 (필수)
  - 스테이크/지방부 선택 (필수)
  - 와드/지부 선택 (필수)
  - 선교 경험 여부 (필수)
  - 추가 정보 (선택사항)

- **지원서 관리**
  - 초안 저장 (DRAFT 상태)
    - 언제든지 수정 가능
    - 여러 번 저장 가능
  - 지원서 제출 (AWAITING 상태)
    - 제출 전 필수 항목 검증
    - 이미 추천서가 있는 경우 제출 불가
    - 제출 후 관리자 검토 전까지 수정 가능

- **지원서 수정**
  - 초안 상태(DRAFT) 또는 대기 상태(AWAITING)인 경우에만 수정 가능
  - 승인(APPROVED) 또는 거부(REJECTED)된 지원서는 수정 불가
  - 상태 확인: 지원서 페이지에서 현재 상태 확인

- **계정 설정**
  - 프로필 정보 확인
  - 비밀번호 변경 (Google 로그인 사용 시 불가)
  - 언어 설정 (한국어/영어)

### 주요 페이지

- **지원서 페이지** (`/application`)
  - 지원서 작성, 수정, 제출
  - 지원서 상태 확인

- **계정 설정** (`/account-settings`)
  - 개인 정보 및 계정 관리

### 사용 흐름

1. **회원가입 또는 로그인**
   - 일반 로그인 또는 Google SSO 로그인
   - 처음 로그인 시 프로필 완성 필요

2. **프로필 완성**
   - 역할 선택: 지원자
   - 스테이크/지방부 및 와드/지부 선택

3. **지원서 작성**
   - 지원서 페이지 접근
   - 필수 정보 입력
   - 선택사항 입력
   - 초안 저장 또는 바로 제출

4. **지원서 제출**
   - 제출 버튼 클릭
   - 시스템 검증 (이미 추천서가 있는지 확인)
   - 제출 성공 시 상태가 `AWAITING`으로 변경
   - 관리자 검토 대기

5. **결과 확인**
   - 지원서 페이지에서 상태 확인
   - 승인/거부 결과 확인
   - 승인/거부 후에는 수정 불가

## 감독 (Bishop)

### 승인 전 (Pending 상태)

#### 접근 가능한 기능

- **대기 페이지 확인**
  - 관리자 승인 대기 중임을 확인
  - 승인 전까지 제한된 기능만 사용 가능

- **추천서 작성** (제한적)
  - 추천서 초안 작성 및 저장 가능
  - 승인 전에는 제출 불가

#### 주요 페이지

- **리더 액세스 대기** (`/leader/pending`)
  - 승인 대기 상태 안내

- **추천서 관리** (`/leader/recommendations`)
  - 추천서 작성 및 초안 관리

### 승인 후 (Approved 상태)

#### 접근 가능한 기능

**리더 대시보드**
- **통계 확인**
  - 초안 추천서 수
  - 검토 대기 중 추천서 수
  - 승인된 추천서 수
  - 거부된 추천서 수
  - 스테이크 내 지원서 수
  - 오늘 생성된 추천서 수
  - 오늘 생성된 지원서 수

- **차트 확인**
  - 스테이크/와드별 분포 (막대 차트)
  - 성별 분포 (원형 차트)

- **빠른 액션**
  - 새 추천서 작성 버튼

**추천서 관리**
- **추천서 작성**
  - 새 추천서 작성 (독립적으로)
  - 지원서에서 추천서 작성 (지원서 정보 자동 입력)
  - 이메일은 선택사항
  - 추천서 초안 저장 (DRAFT 상태)
  - 추천서 제출 (SUBMITTED 상태)
    - 지원서에 대한 추천서는 자동으로 `SUBMITTED` 상태로 제출
    - 독립 추천서는 초안 저장 후 나중에 제출 가능

- **추천서 수정 및 관리**
  - 본인이 작성한 DRAFT 상태 추천서만 수정/삭제 가능
  - SUBMITTED 상태 추천서는 제출 취소만 가능
  - APPROVED/REJECTED 상태 추천서는 수정 불가

- **추천서 목록 조회**
  - 본인 와드 관련 추천서만 표시
  - 전체, 초안, 제출됨, 승인, 거부 탭으로 필터링
  - 지원서와 추천서가 통합된 목록으로 표시
  - 연결된 추천서는 별도 항목으로 표시되지 않음

**지원서 확인 및 추천**
- **지원서 목록 확인**
  - 본인 와드의 지원서만 표시
  - "지원됨" 태그로 구분
  - 지원서와 추천서가 연결된 경우 "추천됨" 태그도 함께 표시

- **지원서 상세 정보 확인**
  - 지원자의 모든 정보 확인
  - 지원서 상태 확인
  - 연결된 추천서 확인 (있는 경우)

- **추천서 작성**
  - 지원서 상세 화면에서 "추천하기" 버튼 클릭
  - 지원서 정보가 자동으로 추천서 폼에 채워짐
  - 추가 정보 입력 후 제출
  - 추천서는 자동으로 해당 지원서와 연결됨

**지도자 코멘트**
- **코멘트 작성 및 관리**
  - 추천서에 코멘트 추가, 수정, 삭제
  - 지원서에 코멘트 추가, 수정, 삭제
  - 본인이 작성한 코멘트만 수정/삭제 가능
  - 다른 지도자의 코멘트는 읽기 전용

- **코멘트 특징**
  - 지원자에게는 보이지 않음
  - 지도자 간 정보 공유용
  - 데스크톱: 상세 화면에서 직접 확인 및 작성
  - 모바일: "지도자 코멘트" 버튼 클릭 시 팝업으로 표시

**계정 설정**
- 프로필 정보 확인 및 수정
- 스테이크/와드 변경 요청 (승인 필요)
- 비밀번호 변경

#### 주요 페이지

- **리더 대시보드** (`/leader/dashboard`)
  - 통계 및 차트 확인
  - 새 추천서 작성 버튼

- **추천서 관리** (`/leader/recommendations`)
  - 전체, 초안, 제출됨, 승인, 거부 탭으로 필터링
  - 지원서와 추천서 통합 목록
  - 데스크톱: 좌측 목록, 우측 상세 정보
  - 모바일: 카드 형태로 표시

### 사용 흐름

1. **회원가입 또는 로그인**
   - 일반 로그인 또는 Google SSO 로그인
   - 처음 로그인 시 프로필 완성 필요

2. **프로필 완성**
   - 역할 선택: 감독
   - 스테이크/지방부 및 와드/지부 선택

3. **관리자 승인 대기**
   - 리더 액세스 대기 페이지에서 승인 대기
   - 승인 전에는 제한된 기능만 사용 가능
   - 추천서 초안 작성은 가능하나 제출 불가

4. **승인 후 리더 대시보드 접근**
   - 관리자 승인 시 자동으로 리더 대시보드로 이동
   - 모든 기능 사용 가능

5. **지원서 확인 및 추천서 작성**
   - 리더 대시보드에서 통계 확인
   - 추천서 관리 페이지에서 지원서 목록 확인
   - 지원서 상세 정보 확인

## 스테이크 회장 (Stake President)

### 승인 전 (Pending 상태)

#### 접근 가능한 기능

- **대기 페이지 확인**
  - 관리자 승인 대기 중임을 확인
  - 승인 전까지 제한된 기능만 사용 가능

- **추천서 작성** (제한적)

  - 승인 전에는 제출 불가

#### 주요 페이지

- **리더 액세스 대기** (`/leader/pending`)
  - 승인 대기 상태 안내

- **추천서 관리** (`/leader/recommendations`)
  - 추천서 작성 및 초안 관리

### 승인 후 (Approved 상태)

#### 접근 가능한 기능

**리더 대시보드**
- **통계 확인**
  - 초안 추천서 수
  - 검토 대기 중 추천서 수
  - 승인된 추천서 수
  - 거부된 추천서 수
  - 스테이크 내 지원서 수
  - 오늘 생성된 추천서 수
  - 오늘 생성된 지원서 수

- **차트 확인**
  - 스테이크/와드별 분포 (막대 차트)
  - 성별 분포 (원형 차트)

- **빠른 액션**
  - 새 추천서 작성 버튼

**추천서 관리**
- **추천서 작성**
  - 새 추천서 작성 (독립적으로)
  - 지원서에서 추천서 작성 (지원서 정보 자동 입력)
  - 이메일은 선택사항
  - 추천서 초안 저장 (DRAFT 상태)
  - 추천서 제출 (SUBMITTED 상태)
    - 지원서에 대한 추천서는 자동으로 `SUBMITTED` 상태로 제출
    - 독립 추천서는 초안 저장 후 나중에 제출 가능

- **추천서 수정 및 관리**
  - 본인이 작성한 DRAFT 상태 추천서만 수정/삭제 가능
  - SUBMITTED 상태 추천서는 제출 취소만 가능
  - APPROVED/REJECTED 상태 추천서는 수정 불가

- **추천서 목록 조회**
  - 본인 스테이크 전체의 추천서 표시 (와드 제한 없음)
  - 전체, 초안, 제출됨, 승인, 거부 탭으로 필터링
  - 지원서와 추천서가 통합된 목록으로 표시
  - 연결된 추천서는 별도 항목으로 표시되지 않음

**지원서 확인 및 추천**
- **지원서 목록 확인**
  - 본인 스테이크 전체의 지원서 표시 (와드 제한 없음)
  - "지원됨" 태그로 구분
  - 지원서와 추천서가 연결된 경우 "추천됨" 태그도 함께 표시

- **지원서 상세 정보 확인**
  - 지원자의 모든 정보 확인
  - 지원서 상태 확인
  - 연결된 추천서 확인 (있는 경우)

- **추천서 작성**
  - 지원서 상세 화면에서 "추천하기" 버튼 클릭
  - 지원서 정보가 자동으로 추천서 폼에 채워짐
  - 추가 정보 입력 후 제출
  - 추천서는 자동으로 해당 지원서와 연결됨

**지도자 코멘트**
- **코멘트 작성 및 관리**
  - 추천서에 코멘트 추가, 수정, 삭제
  - 지원서에 코멘트 추가, 수정, 삭제
  - 본인이 작성한 코멘트만 수정/삭제 가능
  - 다른 지도자의 코멘트는 읽기 전용

- **코멘트 특징**
  - 지원자에게는 보이지 않음
  - 지도자 간 정보 공유용
  - 데스크톱: 상세 화면에서 직접 확인 및 작성
  - 모바일: "지도자 코멘트" 버튼 클릭 시 팝업으로 표시

**계정 설정**
- 프로필 정보 확인 및 수정
- 스테이크/와드 변경 요청 (승인 필요)
- 비밀번호 변경

#### 주요 페이지

- **리더 대시보드** (`/leader/dashboard`)
  - 통계 및 차트 확인
  - 새 추천서 작성 버튼

- **추천서 관리** (`/leader/recommendations`)
  - 전체, 초안, 제출됨, 승인, 거부 탭으로 필터링
  - 지원서와 추천서 통합 목록
  - 데스크톱: 좌측 목록, 우측 상세 정보
  - 모바일: 카드 형태로 표시

### 사용 흐름

   - 일반 로그인 또는 Google SSO 로그인

## 상태 설명

### 지원서 상태

- **DRAFT**: 초안 상태, 수정 가능
- **AWAITING**: 제출됨, 관리자 검토 대기 중
- **APPROVED**: 승인됨, 수정 불가
- **REJECTED**: 거부됨, 수정 불가

### 추천서 상태

- **DRAFT**: 초안 상태, 작성자만 수정/삭제 가능
- **SUBMITTED**: 제출됨, 관리자 검토 대기 중, 작성자만 제출 취소 가능
- **APPROVED**: 승인됨, 수정 불가
- **REJECTED**: 거부됨, 수정 불가

### 리더 승인 상태

- **PENDING**: 관리자 승인 대기 중
- **APPROVED**: 승인됨, 모든 기능 사용 가능

---

## 주의사항

### 지원서 관련
1. **지원서 수정**: DRAFT 또는 AWAITING 상태에서만 수정 가능
2. **이미 추천서가 있는 경우**: 지원서 제출 시 시스템에서 자동으로 확인하고 제출 차단
3. **상태 변경 후**: APPROVED 또는 REJECTED 상태로 변경되면 수정 불가

### 추천서 관련
1. **추천서 수정**: 본인이 작성한 DRAFT 상태만 수정/삭제 가능
2. **추천서 제출 취소**: SUBMITTED 상태인 경우에만 제출 취소 가능
3. **연결된 추천서**: 지원서와 연결된 추천서는 별도 항목으로 표시되지 않음
4. **이메일 선택사항**: 추천서 작성 시 이메일은 필수가 아님

### 리더 관련
1. **리더 승인**: 감독, 스테이크 회장, 대회 지도자는 관리자 승인 후 모든 기능 사용 가능
2. **승인 전 기능**: 승인 전에는 추천서 초안 작성만 가능, 제출 불가
3. **스테이크/와드 변경**: 리더의 경우 변경 시 승인 필요

### 지도자 코멘트
1. **접근 권한**: 지원자에게는 보이지 않으며, 지도자 간 공유용
2. **수정 권한**: 본인이 작성한 코멘트만 수정/삭제 가능
3. **모바일 UI**: 모바일에서는 팝업으로 표시

### 계정 관련
1. **Google SSO**: Google 로그인 사용 시 비밀번호 변경 불가
2. **역할 변경**: 관리자는 사용자 역할을 변경할 수 있으나, 본인의 역할은 변경 불가
3. **사용자 삭제**: 삭제 시 관련 지원서 및 추천서도 함께 삭제됨
